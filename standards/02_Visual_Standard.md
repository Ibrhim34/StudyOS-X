# StudyOS X Visual Standard

## Purpose

This standard defines when and how visual elements should be used to improve learning.

Visuals are educational tools.

They are never decoration.

---

# Core Principle

Every visual must increase understanding.

If a visual does not make learning easier, it should not exist.

---

# Visual Selection Guide

Choose the visual based on the learning objective.

| Learning Need | Recommended Visual |
|---------------|--------------------|
| Compare concepts | Comparison Table |
| Explain a sequence | Flowchart |
| Show a process | Process Diagram |
| Show progression over time | Timeline |
| Organize categories | Tree Diagram |
| Summarize relationships | Concept Map |

---

# Comparison Tables

Use comparison tables when students must distinguish between two or more concepts.

A good comparison table should:

- Compare meaningful attributes.
- Use consistent terminology.
- Highlight important differences.
- Remain concise.

Avoid large tables with unnecessary details.

---

# Flowcharts

Use flowcharts when explaining:

- Algorithms
- Decision processes
- Step-by-step procedures
- Learning pipelines

Flowcharts should always move in one clear direction.

---

# Timelines

Use timelines only when time or historical order is important.

Examples:

- Development of a technology
- Evolution of a concept
- Historical events

---

# Concept Maps

Use concept maps when relationships are more important than sequence.

Examples:

- Related concepts
- Dependencies
- Hierarchies

---

# Visual Quality Rules

Every visual should be:

- Simple
- Clear
- Relevant
- Accurate
- Easy to read

Never overload a visual with information.

---

# Accessibility

Every visual must be understandable without relying on color alone.

Labels should always explain the meaning.

---

# Things to Avoid

Never create visuals:

- Just to make the page look better.
- Without educational value.
- That repeat information already obvious in the text.
- That increase cognitive load.

---

# Success Criteria

A student should understand the concept faster because of the visual.

The visual must reduce effort, not increase it.
